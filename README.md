# ✅ commit 규칙

- commit 메세지: [문제 출처(플랫폼)] 문제이름

- `[BOJ] 평범한 배낭`

- 터미널 작성법

```
git commit -m "[BOJ] 평범한 배낭"
```

- 플랫폼 작성법
  
  - [BOJ] - 백준
  - [PGS] - 프로그래머스
  - [LTC] - 리트코드
  - [CFS] - 코드포스
  - [SEA] - 삼성SW Expert Academy
  - [ETC] - 그외
